## Path of Exile 2
- Full support for Path of Exile 2 coming soon... :)

## Improvements
- Reordering UI elements now uses click & drag
- Many small UI improvements

## Cloud Gaming
- Some support has been added for GeForce Now, other platforms upon request
- Trade features that require parsing chat DO NOT work (trade popups, chat scanner)
- All other features work (hotkeys, cheat sheets, searching)

# Developer Notes
- The entire code base has been cleaned up & commented
- Building & releasing new versions has been automated with GitHub Actions+

## How to Install
- Make sure you have [Java](https://www.java.com) installed
- Download and run 'SlimTrade.jar' from below

*Having issues? Check the [troubleshooting guide](https://github.com/zmilla93/SlimTrade/wiki/Troubleshooting).*
